装备

欢迎来到我的使用装备页面，这里记录了我日常中用到的硬件与软件。

工作台

LG 27UL500

作为外接显示器，效果中规中矩，写代码足够而已

Bose QC35 II

尝试的第一款降噪耳机

MacBook Pro (15-inch)

目前主力开发机，有些年月了

IKBC Poker II & Magic Keyboard

前者是使用的第一款机械键盘，服役中, 后者主要是想体验一下

Magic Mouse 2 - Silver

连同 Magic Keyboard 一起体验了

Apple Pencil

用来画画，学习中

开发

Sublime Text 3

目前主力开发编辑器，开启 Vintage Mode 后多光标处理文本的操作在众多 Vim 模拟环境中是最友好的

Sublime Merge

Git Client, 常用来查看 Git 分支和 Blame 一些文件

VS Code

在前端项目中用的比较多，开箱即用的代码提示和格式化等特性比 ST3 优秀，生态圈繁荣昌盛

Vim/Gvim/MacVim

最早用到的编辑器，极其受到该光标移动方式影响，以至于让我在每个开发环境中都要开启 Vim 模拟器

PHPStorm

开发 PHP 必不可少的 IDE，没有 Editor 和 IDE 的争论，主要看个人对项目和框架的熟悉程度，对于陌生的项目会选择用

Sequel Pro

macOS 下使用频率比较高的数据库管理工具

TablePlus

发展和更新都比较迅速的数据库管理工具，支持 PostgreSQL，MySQL, Redis, SQLite 等多种数据库

iTerm 2

macOS 下必备系列

Postman

发起 HTTP 请求常用的工具，设置环境变量，生成代码，调试等功能应有尽有

Altair

尝试过众多的 GraphQL 客户端和 Playground 后最终选择的一款，设置变量，压缩和美化查询等大家有的功能都有，比较出众的特性是上传文件和从文档中能直接生成查询

Oh My Zsh

用过的都说好

Alfred

天天用系列，快速切换应用，一些插件更能提高生产力

Solis

一款为开发者和设计师而生的浏览器，能模拟多种设备的显示，实为调试 UI 时不可缺少的神器。竞品 Sizzy 拥有更多的支持者，UI 似乎也更胜一筹，但暂时无法接受一个工具还要设计成订阅制的方式来购买和使用

主题/字体

Material Theme

普及范围比较广的一款主题，支持的编辑器和 IDE 也非常多

Nord Theme

一款小清新的主题，打开它的官网，相信你会喜欢

Rainglow

众多的 Color Scheme 可供选择

Fira Code

用得最早的一款字体，开源，使用群体也很庞大

Operator Mono

最开始在 Jeffrey Way 的视频中看到，也看到很多人求其名字的字体，价格感人

Dank Mono

同样是很多人询问名字的字体，最有特点的是字母 f 的设计

Cascadia Code

最近在用的字体，star 数也很高

摄影/设计

Canon 6D

LanternD 推荐的机子，感谢极具价值的建议，庆幸我入门摄影的机子就是 6D，也陪伴了我四年的时光，拍出一些满意的照片，美丽仍在继续

百诺 (Benro) 三脚架 it25

陪伴我去过很多地方，拍摄延时的照片，功不可没

莱思 (Lacie) 移动硬盘

主要存储照片和视频

Figma

设计和排版原型稿，制作基本 SVG 会用到

Procreate

iPad 上的一款手绘 app，在学习用 Apple Pencil 创作出自己满意的作品

Dribbble

非常喜欢里面大部分的作品，给了我很多的灵感

Pinterest

可以作为一个素材来源，其 More like this 版块的关联素材总能让你有更多精彩的发现

QuickTime

录制屏幕时用自带的 QuickTime 能满足了

ScreenFlow

编辑视频时会用，目前用的并不多

服务

Notion

目前仅做 TODO List 来用

SVGOMG

压缩 SVG 常用的在线服务

推荐资源

DesignCode

里面的设计非常赞

Laracasts

Laravel 社区学习资源的不二之选，还有前端，测试等领域丰富的知识

Raywenderlich

App 开发不可多得的资源，视频，书籍，文章等都值得一看
